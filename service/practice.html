Обработка аттестационной ведомости

Авторы: Чиркин Александр, Шурыгина Ангелина.

Используемые технологии. php, MySql, html, css.

Приложение позволяет преподавателям размещать результаты текущих аттестаций. А студентам просматривать их и контролирровать свою успеваемость. В приложении будут реализованы 2 роли. Преподаватель и студент. Они будут определяться при авторизации.

Функции преподавателя.

  • Загрузка аттестационной ведомости(файла в формате excel)
  • Редактирование ведомости на сайте.
  • Сортировка студентов в группе по успеваемости.
  • Просмотр среднего балла успеваемости студентов в группе

Функции студента

  • Просмотр своих баллов по аттестациям
  • Просмотр своего среднего балла по аттестациям и сравнение со средним баллом группы
  • Сортировка предметов студента в порядке повышения(понижения) среднего балла по каждому из них
  • В начале каждого семестра студент может выставить себе желаемые оценки по каждому предмету. После очередной аттестации студент будет видеть совпадают ли результаты его аттестаций с теми оценками которые он хочет получить(например выделение цветом предметов)

План действий:

  1. Составить er-диаграмму базы данных проекта.
  2. Создать базу данных используя MySQL.
  3. Сделать страницу регистрации, авторизации.
  4. Функционал загрузки и редактироваания ведомости (для преподавателя).
  5. Функционал просмотра результатов аттестаций.
  6. Создание дизайна приложения(html, css).
